# CodeAlpha Basic Network Sniffer

## Description

A Python-based network packet sniffer developed as part of the CodeAlpha Cyber Security Internship. The project uses Scapy to capture and analyze live network traffic.

## Features

* Capture live network packets
* Display source and destination IP addresses
* Detect TCP, UDP, and ICMP protocols
* Show source and destination port numbers

## Technologies Used

* Python 3
* Scapy
* Kali Linux
* Git & GitHub

## Installation

```bash
pip3 install -r requirements.txt
```

## Run the Project

```bash
sudo python3 network_sniffer.py
```
